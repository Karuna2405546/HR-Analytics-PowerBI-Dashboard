# HR Analytics Dashboard – Employee Attrition Analysis

This project presents an end-to-end HR Analytics Dashboard built using Power BI to analyze employee attrition trends and workforce insights.

## 📌 Key Features
- Interactive KPIs (Total Employees, Attrition Rate, Avg Salary, Avg Age)
- Department-wise & Job Role-wise attrition analysis
- Gender, Age Group & Education Field analysis
- Business-driven insights for HR decision making

## 🛠 Tools & Technologies
- Power BI
- DAX
- Data Modeling
- Data Visualization
